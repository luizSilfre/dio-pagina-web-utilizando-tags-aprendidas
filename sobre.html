Sobre Este Projeto

Este site foi desenvolvido como parte do desafio da Formação HTML Web Developer da DIO.

O objetivo é aplicar na prática os conceitos aprendidos nas aulas iniciais de HTML, utilizando as principais tags de estruturação, formatação de texto, listas e links.

Mais do que apenas cumprir um requisito técnico, este projeto representa:

  • A consolidação dos primeiros fundamentos do desenvolvimento web;
  • A prática da organização semântica do conteúdo;
  • O início da construção de uma base sólida em front-end.

Cada seção deste site apresenta, com minhas próprias palavras, o que aprendi até aqui, desde a estrutura básica de uma página HTML até o uso de atributos, listas e links.

Estou utilizando este desafio como forma de fixar o conhecimento, praticar boas práticas e evoluir gradualmente na jornada como desenvolvedor web.

Este é apenas o começo.
