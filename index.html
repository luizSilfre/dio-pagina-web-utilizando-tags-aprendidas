Inspetor de Elementos

O Inspetor de Elementos é uma ferramenta presente em todos os navegadores modernos.

Ele permite:

  1. Visualizar a estrutura HTML da página;
  2. Testar alterações em tempo real;
  3. Identificar problemas de layout;
  4. Experimentar estilos CSS sem alterar o código original.

Essa ferramenta é extremamente útil para entender como uma página foi construída e para corrigir erros com mais rapidez.

Estrutura Básica do HTML

De forma simples, uma página HTML é composta por três grandes blocos:

  1. <html>: Define que o arquivo é um documento HTML;
  2. <head>: Contém informações e configurações da página;
  3. <body>: Contém todo o conteúdo visível para o usuário.

Por convenção, a página principal se chama index.html.

Dentro da tag <head>, encontramos por exemplo a tag <title>, que define o título da página exibido na aba do navegador.

Já a tag <body> é onde construímos o conteúdo real da página: títulos, textos, listas, links, imagens e muito mais.

Tags

As tags são os elementos que estruturam e organizam o conteúdo de uma página.

A maioria delas funciona em pares:

  • <i> Texto em itálico </i>
  • <strong> Texto em negrito </strong>

Abrimos a tag e depois a fechamos com </...>.

Existem também tags que não precisam de fechamento, como:

  • <input type="text" />
  • <img src="imagem.jpg" />

Essas são chamadas de tags auto-fechadas.

Atributos das Tags

As tags podem receber atributos, que são propriedades que modificam seu comportamento.

Alguns atributos são globais, como:

  • id: Identificador único do elemento;
  • style: Define estilo CSS inline.

Exemplo:

<p id="paragrafo1" style="color: blue;">Texto azul</p>

  • type: Usado na tag <input> para definir o tipo de campo: texto, senha, checkbox, etc;

Textos no HTML

Para organizar textos usamos:

Títulos

Tags de <h1> até <h6>

  • <h1>: Título principal;
  • <h2>: Subtítulo;
  • <h3>: Título de seção;
  • <h4>: Título de subseção;
  • <h5>: Título de sub-subseção;
  • <h6>: Título de sub-sub-subseção.

Parágrafos

Usamos a tag <p> para criar parágrafos de texto.

Citações

Usamos a tag <blockquote> para criar citações.

Destaques dentro do texto

Podemos usar dentro de um <p>:

  • <strong>: Texto em negrito;
  • <em>: Texto em itálico;
  • <u>: Texto sublinhado;
  • <mark>: Texto destacado com fundo amarelo;

Listas Ordenadas e Não Ordenadas

HTML possui dois tipos principais de listas:

Lista Ordenada (numerada)

Usamos a tag <ol> para criar uma lista ordenada, onde os itens são numerados automaticamente.

Lista Não Ordenada (com marcadores)

Usamos a tag <ul> para criar uma lista não ordenada, onde os itens são marcados com bullets (pontos).

Em ambos os tipos de listas, cada item é definido com a tag <li>.

Também é possível criar sublistas, colocando uma <ol> ou <ul> dentro de um <li>.

Links

Os links são definidos pela tag <a>.

Exemplo:

<a href="https://www.dio.me">Link para a DIO</a>

Que retorna:

Link para a DIO

Principais atributos:

  • href: Define o destino do link;
  • target: Define onde o link será aberto;
  • title: Define um título para o link.

Valores comuns de target:

  • _blank: Abre o link em uma nova aba;
  • _self: Abre o link na mesma aba (padrão);
  • _parent: Abre o link na janela pai;
  • _top: Abre o link na janela principal.

Boa prática:

  • Links externos devem usar: _blank;
  • Links internos devem usar: _self.
